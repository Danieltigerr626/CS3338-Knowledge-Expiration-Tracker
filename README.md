Project Goals
The Personal Knowledge Expiration Tracker is designed to address the "Forgetting Curve"—the natural decline of memory retention over time. In a fast-paced academic and professional environment, information is often lost shortly after it is acquired. This software matters because it transitions "temporary learning" into "permanent knowledge" by using automated reminders to prompt active recall at the exact moment a topic is likely to be forgotten.


The project aims to achieve the following core objectives:

Automated Study Tracking: Develop a database-driven system to log learning sessions and calculate "expiration dates" for specific topics based on spaced-repetition algorithms.

Dynamic Calendar Integration: Implement a visual calendar interface that allows users to see an overview of upcoming review deadlines and manage their study workload.

Active Recall via Flashcards: Build a built-in testing tool that enables users to verify their retention of specific "expired" topics through digitized flashcards.

Containerized Portability: Ensure the entire software stack (Frontend, Backend, and Database) is deployable via Docker for consistent performance across any user environment.

Quality Assurance & Reporting: Maintain high software reliability through documented test runs in TestRail and task management via 1-month Jira sprints.
